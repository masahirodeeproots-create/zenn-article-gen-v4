## style_guide.mdの学習（注目）

今回のPDCAを通じて、style_guide_updaterが**新しいルールを自動学習**:

### IMPORTANT Rulesに追加されたルール
- `[style]` HARD FAIL: です・ます比率≥80%チェック — 箇条書き行末も対象
- `[voice]` 概念フレームの語は冒頭3段落以内に登場させる

### Learned Rulesに追加
- `[style]` 箇条書き行末もです・ます体で統一
- `[structure]` 複数エージェント列挙を1エージェント視点の散文に変換
- `[structure]` 3幕の構造スタイルを差別化（未解決型/逆算型/交差型）

### Failure Patternsに追加
- `[style]` 同形文末「〜しました」4文以上連打でリズム単調化
- `[structure]` 3幕全て「失敗→解決策」構造だと幕3が読める

**→ 次回実行時、これらのルールがWriter/Reviewerに自動適用される。蓄積基盤が機能している。**
